What It Measures

The Flood Risk Score checks whether a property falls within officially designated flood overlays in state government planning schemes. A higher score means lower flood risk.

0-20Very High Risk
21-40High Risk
41-60Moderate
61-80Low Risk
81-100Very Low Risk

Scoring Logic

  1. State detection — Identify the Australian state from coordinates using bounding boxes.
  2. Overlay query — Query the relevant state's ArcGIS REST API for flood planning overlays at the point.
  3. Severity classification — Classify by severity: Floodway (most severe) to Land Subject to Inundation (moderate).
  4. Score assignment — Multiple overlapping zones take the worst (lowest) score.

Coverage

Currently covers VIC, NSW, SA, TAS and ACT. QLD, WA and NT planning overlay data is not yet available via public ArcGIS services. Locations in unsupported states return a default score of 85 with a note.
