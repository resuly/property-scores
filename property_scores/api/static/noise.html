What It Measures

The Noise Score rates how quiet a location is on a scale of 0 to 100, where 100 is the quietest. It estimates the cumulative traffic noise from all roads within 1 km using the CRTN (Calculation of Road Traffic Noise) simplified formula and Overture Maps road classification data.

Road Class to Traffic Volume Mapping

Overture Maps classifies every road segment (motorway, trunk, primary, secondary, tertiary, residential). Each class maps to a reference noise level based on typical AADT (Annual Average Daily Traffic):

Calculation Steps

  1. Find all roads within 1 km of the target coordinate using DuckDB spatial queries on Overture parquet data.
  2. Per-road noise contribution: L = L_ref - 10 log10(d / 13.5) - 3.0 where d = distance in meters and 3.0 dB is ground absorption.
  3. Energy summation across all roads: L_total = 10 log10(sum of 10^(Li/10))
  4. Convert to score: score = clamp(100 - (L_total - 35) x 1.5, 0, 100)
