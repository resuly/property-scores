What It Measures

The Noise Score rates how quiet a location is on a scale of 0 to 100, where 100 is the quietest. It combines three noise sources (road traffic, rail/tram, aircraft) using physics-based acoustic models and real government data.

Noise Sources

1. Road Traffic

Uses VicRoads AADT (Annual Average Daily Traffic) ground truth for 14,600+ monitored road segments in Victoria. For unmonitored streets, falls back to conservative estimates based on Overture Maps road classification.

2. Train & Tram

Uses PTV GTFS timetable data with actual service frequencies per route. SEL-based noise model accounts for peak vs off-peak service rates, pass-by duration, and distance attenuation.

3. Aircraft

Queries VicPlan planning overlays for MAEO (Melbourne Airport) and AEO (regional airports) zones. Maps ANEF contours to noise penalties based on AS2021-2015.

Propagation Model

  1. Road emission — CRTN L10: 42.2 + 10 log10(AADT) with duty-cycle correction for low-traffic roads
  2. Rail emission — SEL-based: L_peak + 10 log10(duration) + 10 log10(N/3600) using actual PTV service counts
  3. Distance attenuation — 10 log10(d / d_ref) + 3 dB ground absorption + 0.06 dB/m excess above 50m
  4. Building screening — Maekawa barrier formula using Overture building heights, up to 20 dB reduction
  5. Energy summation — Top-3 road + top-2 rail sources combined: L_total = 10 log10(sum 10^(Li/10))
  6. Aircraft penalty — ANEF zone adds ambient noise from flight path exposure

Deduplication

Roads appearing in both VicRoads AADT and Overture Maps are deduplicated by distance proximity (80m threshold) to prevent double-counting the same physical road.

Limitations

  • VicRoads AADT only covers VIC state-monitored roads (arterials/highways)
  • PTV GTFS covers Melbourne metro only; no interstate rail
  • Aircraft overlays currently VIC only (MAEO + AEO); no NSW/QLD ANEF
  • Building screening only effective for secondary noise sources (nearest road usually has line-of-sight)
  • No industrial or construction noise sources
