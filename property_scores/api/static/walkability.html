What It Measures

The Walkability Score rates how convenient daily errands are on foot, on a scale of 0 to 100. It checks proximity to 13 essential amenity categories and applies a distance decay: facilities within a 5-minute walk score highest, while anything beyond 20 minutes scores zero.

90-100Walker's Paradise
70-89Very Walkable
50-69Somewhat Walkable

Calculation Steps

  1. Search POIs within 1.5 km of the target coordinate using Overture places data.
  2. Classify each POI into one of 13 categories by keyword matching on Overture category labels.
  3. Distance decay per category: decay = max(0, 1 - distance_m / 1500). Only the nearest facility in each category counts.
  4. Weighted sum: score = sum(weight_i x decay_i) / sum(weights) x 100
